

## switch语句相关

请注意：switch语句的表达式必须是整型（int, char, enum等）或可以隐式转换为整型的类型

形如

```C
float w;	
switch(w)
	{ 
		case 1.0: printf("*\n");
		case 2.0: printf("**\n");
	}
```

出现`w`、`1.0`、`2.0`等浮点数与浮点数常量是不被允许的！

### switch()能完全替代if吗？

当条件判断是单一变量的`等值判断`时（判断某个变量是否等于若干固定值，比如`a=200`），switch能替代多个`if-else`分支。(需要注意的是`a>200`这种虽然不报错，但是逻辑上确实不如用if好)

但需要判断`范围条件`（如`if(x > 200)`）或者`区间判断`

（如`if(x > 5 && x < 10)`）等复杂逻辑时，switch不能替代



### default相关

1.default不需要可以省略

2.default可以放置在任意位置，但只有放在最后可以省略break，如果放其他位置必须带上break
3、case下面可以{}包裹case下的语句，语法不会报错，增加代码可阅读性

4.C/C++下，case下面不能定义变量



## for循环相关

i = 0 初始化表达式：第一次执行循环

i < 0 用于判断决定何时进入循环，何时离开结束循环

i++ 迭代语句：每一次循环后必须执行的语句

循环次数：条件表达式值~初始化表达式值

#### 面试真题

```c
for(printf("a"); printf("b"); printf("c"))
{
    printf("d");
}
```

是否报错？若报错，报错是什么问题；若不报错，那会输出什么样的结果？

————————————————

**初始化表达式**：可以是任何有效的C表达式，甚至与循环无关

```C
for(printf("Hello"); ...; ...)  // 输出一次Hello
for(a=1, b=2; ...; ...)        // 多个表达式用逗号分隔
for(; ...; ...)                // 也可以为空
```

**条件表达式**：可以是任何返回值的表达式

- 除 `0`、`NULL`、`false` 外都视为真

- `printf()` 返回输出的字符数，非0即为真(
  如

  1.`printf("")`，条件为假，因为空字符串返回0

  2.`printf()`，不能单独写 `printf()`，必须提供格式字符串

  3.`printf("0")`，返回值1，条件为真

  )

- 如果为空，默认为真（无限循环）



**更新表达式**：同样可以是任何表达式

```C
for(...; ...; printf("c"), i++)  // 多个操作
for(...; ...; )                  // 也可以为空
```

所以上述例子永远不会停止，因为判断条件是`printf("b")`，为真， 除第一次是adbc外，后续都是dbc无限循环，即adbc dbc dbc......
